# Donation Management System

A console-based application built in C language using Dev-C++.

## Features
- Add donations (Monetary, Health, Miscellaneous)
- View donation history by donor name
- View donations sorted by date
- View total donations and amount
- Auto-award certificates based on donation type

## How to Run
1. Open in Dev-C++
2. Compile and run main.c
3. Make sure organisations.txt is in the same folder

## Language
C (Standard C99)
